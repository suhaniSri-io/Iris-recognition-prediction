# 🌸 Iris Flower Species Prediction

## Overview

This project demonstrates a simple Machine Learning workflow for classifying iris flowers into their respective species using the K-Nearest Neighbors (KNN) algorithm. The model is trained on the famous Iris dataset provided by Scikit-learn and predicts flower species based on their measurements.

## Features

* Uses the built-in Iris dataset from Scikit-learn
* Implements the K-Nearest Neighbors (KNN) classification algorithm
* Splits data into training and testing sets
* Evaluates model performance using accuracy score
* Predicts the species of new iris flower samples
* Visualizes the dataset using scatter plots

## Dataset

The Iris dataset contains 150 samples of iris flowers with four features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target Classes:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## Technologies Used

* Python
* Jupyter Notebook
* Scikit-learn
* Pandas
* NumPy
* Matplotlib

## Project Workflow

1. Load the Iris dataset
2. Explore and visualize the data
3. Split the dataset into training and testing sets
4. Train a K-Nearest Neighbors (KNN) classifier
5. Evaluate model accuracy
6. Predict the species of a new iris flower sample

## Results

The trained KNN model successfully classifies iris flowers with high accuracy, demonstrating the effectiveness of machine learning for classification tasks.
